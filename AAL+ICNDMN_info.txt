This node list contains the centroid of each region in the Automated Anatomical Labelling atlas, plus known Default Mode Network regions.
This list is ideal for source-based resting state MEG analysis.

Notes:
  - Nodes from de Pasquale were mirrored onto the opposite hemisphere where given in only one hemisphere.
